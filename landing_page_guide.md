# 표준주소실록 (Standard Address Annals) 활용 가이드

## 🚀 우리 업무가 이렇게 편해집니다!

**"뒤죽박죽 섞인 주소, 하나하나 찾느라 힘드셨죠?"**
이제 **표준주소실록**으로 1초 만에 해결하세요.

![표준주소실록 포스터](landing_guide_poster_1767965681469.png)

### ✨ 핵심 기능 3가지
1.  **초정밀 주소 변환**: 민원인이 대충 적은 주소도 **정확한 도로명 주소 + 행정동**으로 변환해 드립니다.
2.  **대량 일괄 처리**: 엑셀에 있는 주소 100개, 1000개도 **복사+붙여넣기** 한 번이면 끝!
3.  **한눈에 보는 지도**: 변환된 주소 위치를 **지도 위에 마커**로 표시해 드립니다. (현장 방문 동선 짤 때 최고!)

---

## 📖 3단계 사용법 (누구나 쉽게!)

![앱 사용 예시](app_ui_mockup_1767965712730.png)

### Step 1. 접속하기
웹 브라우저에서 **[gjdong.vercel.app](https://gjdong.vercel.app)** 에 접속하세요.
*(별도 설치 없이 바로 사용 가능합니다)*

### Step 2. 입력하기
변환하고 싶은 주소를 입력창에 넣으세요.
*   **하나만 찾을 때**: 그냥 주소 입력 후 엔터!
*   **여러 개 찾을 때**: 엑셀에서 **[시설명 | 주소]** 두 칸을 드래그해서 복사(Ctrl+C) 후 붙여넣기(Ctrl+V)!

### Step 3. 결과 챙기기
**'주소 검색'** 버튼을 누르면 변환 끝!
*   결과를 클릭해서 복사하거나,
*   **[엑셀 다운로드]** 버튼으로 깔끔하게 파일로 저장하세요.

---

### 💡 꿀팁
*   **오류 해결**: 주소가 이상하면 자동으로 빨간색으로 표시해 드려요.
*   **개인정보 보호**: 입력한 주소는 서버에 저장되지 않고 즉시 변환만 수행하니 안심하세요.
