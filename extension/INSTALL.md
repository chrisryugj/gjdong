# 표준주소실록 브라우저 확장 프로그램

한국 주소를 표준 형식으로 즉시 변환하는 브라우저 익스텐션

---

## 설치 방법

### Chrome
1. 주소창에 `chrome://extensions` 입력
2. 우측 상단 **개발자 모드** 켜기
3. **압축해제된 확장 프로그램을 로드합니다** 클릭
4. `extension/build/chrome-mv3-prod` 폴더 선택

### Edge
1. 주소창에 `edge://extensions` 입력
2. 좌측 하단 **개발자 모드** 켜기
3. **압축 풀린 항목 로드** 클릭
4. `extension/build/chrome-mv3-prod` 폴더 선택

---

## 사용법

### 1. 팝업 (기본)
- 툴바 아이콘 클릭 → 주소 입력 → 변환
- 여러 주소 한번에 붙여넣기 가능 (줄바꿈 구분)
- 변환 결과 클릭 시 클립보드 복사

### 2. 우클릭 메뉴
- 웹페이지에서 주소 텍스트 **드래그 선택**
- 우클릭 → **"표준주소 변환"** 클릭

### 3. 단축키
- `Ctrl+Shift+C` (Mac: `Cmd+Shift+C`)
- 클립보드에 복사된 주소를 즉시 변환

### 4. 자동 감지
- 웹페이지에서 주소를 복사하면 자동 감지 → 알림으로 변환 결과 표시
- 설정에서 ON/OFF 가능

## 출력 형식 (7종)

| 형식 | 예시 |
|------|------|
| 표준형식1 | 광진구 아차산로 400(자양동 870, 자양2동) |
| 표준형식2 | 광진구 아차산로 400(자양동 870) |
| 도로명주소 | 서울특별시 광진구 아차산로 400 |
| 지번주소 | 서울특별시 광진구 자양동 870 |
| 행정동 | 자양2동 |
| 우편번호 | 05006 |
| 세부주소 | 101동 201호 |

## 설정

아이콘 우클릭 → **옵션** (또는 팝업 내 설정 아이콘)

- **API 서버**: 기본 `https://gjdong.vercel.app` (로컬 개발 시 `http://localhost:3000`)
- **기본 출력 형식**: 7종 중 선택
- **지도 제공자**: OpenStreetMap / Kakao / Naver
- **알림**: ON/OFF
- **자동 감지**: ON/OFF
